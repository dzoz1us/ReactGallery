Галерея изображений 

React-приложение для просмотра и навигации по коллекции изображений.

 Функционал приложения
Основные возможности:
1. Просмотр изображений

- Отображение основного изображения в крупном формате

- Поддержка различных форматов изображений (JPG, PNG, etc.)

2. Навигация между изображениями

- Кнопки "Вперед" и "Назад" для последовательного переключения

- Циклическая навигация (после последнего изображения показывается первое)

3. Прямой доступ к изображениям

- Панель миниатюр для быстрого перехода к любому изображению

- Визуальное выделение активного изображения

4. Информационный дисплей

- Счетчик текущего положения (например: "3 / 8")

- Название галереи

Технические особенности:

5. Адаптивный дизайн

- Оптимизация для мобильных устройств

- Гибкая верстка под разные размеры экранов

6. Единообразие отображения

- Все изображения отображаются в контейнерах фиксированного размера

- Сохранение пропорций изображений без искажений

- Стабильный интерфейс без "прыгающих" элементов

7. Интерактивные элементы

- Плавные анимации при наведении
